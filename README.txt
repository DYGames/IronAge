IronAge
Unity - 2020.3.18f1
Windows

아무 버튼을 누르면 게임이 시작된다.

총 10개의 웨이브동안 여러 방향에서 몰려오는 적으로부터 가운데 움막을 지키는게 목표이다.

좌측 하단에 버튼을 눌러 타워 구매, 업그레이드를 진행 할 수 있다.

타워를 구매하고, 맵을 클릭해 타워를 설치 할 수 있다.

좌측 상단에 현재 보유하고 있는 금액이 표시된다.

가운데 상단에 웨이브 정보가 표시된다.

적은 설치된 타워를 먼저 부수고, 마지막으로 타워가 하나도 없으면 움막을 부순다.

도끼 타워, 방패 타워, 창 타워가 있다.

방패 타워는 공격은 할 수 없지만 방어력과 HP가 높다.

공격력, 방어력, 공격 속도를 업그레이드 할 수 있다.

게임이 종료되면 게임을 진행하는 동안의 통계, 랭크가 표시되고 게임을 다시 시작 할 수 있다.
